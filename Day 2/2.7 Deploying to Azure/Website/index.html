MSA FoodBot 2017

Website Instructions:

1. Setup a new git repo
2. Create an ASP.NET Web Application in GitRepo
3. Add new items: html, css and JavaScript
4. Right-click the html file and "Set As Start Page"
5. Link stuff in html file
6. Add the following meta tag to the head
' <meta name="viewport" content="width=device-width, initial-scale=1.0"> '
this gives the browser instructions on how to control the page's dimensions and scaling.
